How I built this website

I created this guide for people who also want to make their own page. I had a lot of struggles, so I want to share them with you and help you have an easier time creating your site.

Let’s get into it. First you want to download Git and nodejs. Always try to download the most recent version. Then simply follow the onscreen instructions.

Once it has finished downloading, you can check to see if you have truly have it. Press the windows button and type “cmd” in the searchbar. This is known as a terminal. You can have multiple of them. When the terminal has been booted up, type

node --version

to check if your computer has node. When you typed this, the terminal recognizes the first word and realizes that this command if associated with node. It then tells node about this command, and node checks if it has this “–version” command. It realizes that it does, and it should gives you something similar to the picture below.

Type 

git -v

to check the version. Once again, you are telling git through the terminal that you are using the “-v” command. Git senses that it is a valid command, and it gives you a response. If your computer has git then the output should be something similar to the picture below.

Once you have done all of that, you are ready to download hexo. Do this by typing the following into the terminal:

npm install -g hexo-cli

There are some things to explain here. Remember node, what you just downloaded? npm means node package manager. That command is telling the node package manager who is responsible for packages to download hexo.

Again, we want to make sure that we have hexo. Type the following to check Hexo’s version:

hexo v

This command isn’t different from the ones we typed above. Now we are asking Hexo for its version. The output should resemble a bunch of words and numbers like below:

Now, you are ready to make a new hexo project. Type the following:

hexo init [project name]

This may take a bit. Hexo is currently initiating a new project. Once it is done, it should say “start blogging with Hexo!” like below

Well you just created a new hexo project! Now, open up files and go to the exact place that your terminal is saying. In my case this is c:/Users/Administrator, so I’ll navigate over there. When you are there, you should see a new folder called exactly what you named your Hexo project. Open it and you should see the following:

Now, yours may not look exactly like mine, because I am using Sublime Text as my editor. When you downloaded git, there should have been a prompt to ask you what editor you are going to use. I picked Sublime Text, but anything else also works fine. Just don’t be surprised that some of my files may look different from yours.

Now let’s get into editing. Ignore the .github, node modules and scaffolds folder for now. We will not be editing those right now. Instead, I want you to open the source folder. Inside there should be another folder named
